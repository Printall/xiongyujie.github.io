linux的文件属性

在linux中，使用ls -l查看l文件属性，具体情况如下

第一个字母代表的是文件的属性：

​ l：表示链接文件

​ -：表示为文件

​ d：表示为目录

​ b：表示装置文件里面的可供存储的设备接口

​ c：表示装置文件里面的串行设备

linux文件的权限包括读，写，执行分别用r, w, x表示

有三种模式修改权限分别为：+，-， =

文件权限的十进制表示r,w,x分别为1，2，4

文件的粒度分别为拥有者，组群，其他分别用u, g, o

a代表所有粒度

文件权限管理
  • chown：改变文件所属用户和组
  • chmod：修改用户权限
  • chgrp：更改文件组属性

chomod [option] [mode] file

option:

​ -c : 显示更改的部分信息

​ -f: 忽略错误符号

​ -h: 修复符号链接

​ -v: 显示详细的处理信息

 `-R: 递归修改`

chgrp [-cfhRv] [所属组群] [文件或者目录]

option：

​ -c: 显示更改信息

​ -f: 不显示错误信息

​ -h: 只对链接的文件修改，不修改其他文件

​ -R: 递归处理

​ -v: 显示指令执行过程

chown [-cfhvR] user[:group] file

​ -c: 显示更改信息

​ -f: 不显示错误信息

​ -h: 只对链接的文件修改，不修改其他文件

​ -R: 递归处理

​ -v: 显示指令执行过程

文件目录管理

常用的处理命令：

命令 功能
ls 列出目录名和文件名
cd 切换目录
pwd 显示当前目录
mkdir 创建文件或者吗目录
rmdir 删除文件或者目录
cp 复制文件或者目录
rm 移除文件或者目录
mv 移除或者移动文件或者目录

常用命令：

ls [option] [目录]

  • -a: 列出所有文件，包括隐藏文件
  • -d: 列出目录本身，而不是内部数据
  • -l: 列出文件或目录的属性

mkdir [option] 目录名称

  • -m: 配置文目录的权限
  • -p: 递归创建

cp: [option] source destination

-a: 等于-pdr

-d: 复制链接属性

-f: 强制复制

-i: 覆盖询问

-l: 硬链接复制

-p: 复制文件的属性

-r: 递归复制

-s: 复制为软连接

-u: 根据源文件和目标文件的版本复制

linux文件内容查看
命令 功能
cat 由第一行开始显示文件内容
tac 由最后一行显示内容
nl 显示行号
more 分页显示内容
less 和more相同，但是可以向前翻页
head 只看前几行
tail 只看后几行

cat [option] file

  • -A：=-vET
  • -b: 列出行号
  • -E: 将结尾断行字节显示
  • -n：列出行号，包括空白
  • -T: 显示table键，并以^|显示
  • -v: 列出特殊字符

nl [option] file

  • -b：指定行号的输出方式; a包括空行输出;b表示不输出空行行号
  • -n：行号的输出位置；ln左边输出行号，rn右边述出行号，rz填充0右边显示
  • -w：行号栏占位的位数

more file

  • 空格键：向下翻页
  • enter：向下翻一行
  • / ：向下搜索
  • :f：显示文档行号，以及当前行号
  • q：退出文档
  • b：往回翻页

less file

  • 空白键：向下翻页
  • pagedown：向下翻页
  • pageup：向上翻页
  • /：向下搜索
  • ?：向上搜索
  • n：重复前一个搜索
  • N：反向(向上或者向下)重复前一个
  • q： 退出文档

head/tail - n ：表示前n行或者后n行显示
