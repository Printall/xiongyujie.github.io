question 4

该操作系统可以支持的最大物理内存量是多少？为什么？

UPAGES = PGSIZE * NPTENTRIES = 4096 * 1024 = 4MB,

size(pageinfo) = 8B,

4MB / 8B * 4096B = 2GB

question 5

如果我们实际拥有最大的物理内存量，那么管理内存有多少空间开销？这个开销如何分解？

UPAGES = 4MB, 页表目录: 1024 * 32 / 8 = 4KB, 页表：4MB / 8B * 32 / 8 = 2MB

一共内存开销6MB + 4KB
