01_环境配置

发表于 2021-04-02 更新于 2021-04-05 分类于 MIT OS 6.828 Disqus：

装好虚拟机可以配置环境了，我这里的用的软件是virtualbox+ubuntu20.4

配置环境：

安装qemu，参考

https://www.qemu.org/download/#source

https://pdos.csail.mit.edu/6.828/2018/tools.html

下载遇见中遇见的错误，

发现错误一：

​ ERROR: GNU make(make) not found

解决方法：

​ sudo apt install make

错误二：

​ ERROR: Cannot find Ninja

解决方法：

​ sudo apt install ninja-build

除此之外，编译qemu还需要下载：libsdl1.2-dev, libtool-bin, libglib2.0-dev, libz-dev, and libpixman-1-dev.

使用sudo apt install [package]即可

创建/home/6.828文件夹，进入到文件夹中，打开命令行，输入

然后进入lab中，使用make && make qemu，到此环境配置成功。
